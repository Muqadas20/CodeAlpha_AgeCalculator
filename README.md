# 🌟 CodeAlpha Age Calculator

## 📌 Project Overview

This project was developed as part of the **CodeAlpha Web Development Internship – Task 1**.

The **Age Calculator** is a web application that allows users to enter their **Date of Birth** and instantly calculates their exact age in **Years, Months, and Days**. It also displays the user's **Zodiac Sign** and the **countdown to their next birthday**.

---

## ✨ Features

* 📅 Select Date of Birth
* 🎂 Calculate age in Years, Months, and Days
* ♈ Detect Zodiac Sign
* ⏳ Display countdown to the next birthday
* ✅ Input Validation
* 🎨 Modern Neon Blue User Interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* DOM Manipulation
* JavaScript Date Object

---

## 🚀 How to Use

1. Open the application in your browser.
2. Select your Date of Birth.
3. Click the **Calculate** button.
4. View your:

   * Age (Years, Months, Days)
   * Zodiac Sign
   * Days remaining until your next birthday
